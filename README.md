# Aplikasi Reservasi Ruang Kelas

Aplikasi sederhana berbasis Java dengan GUI untuk melakukan reservasi ruang kelas.  
Pengguna utama aplikasi ini adalah **dosen** yang harus login terlebih dahulu menggunakan ID dan password sebelum mengakses fitur aplikasi.

---

## Fitur

- Login dengan ID dan password
- Halaman utama (landing page) setelah login berhasil
- Antarmuka grafis yang sederhana dan mudah digunakan

---

1. Sdah punya **Java JDK** versi 8 atau lebih baru.
