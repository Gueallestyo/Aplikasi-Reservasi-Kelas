# 📚 Aplikasi Reservasi Kelas

Aplikasi desktop berbasis Java yang digunakan untuk memudahkan proses reservasi ruang kelas oleh dosen di lingkungan kampus. Dibuat sebagai proyek pembelajaran menggunakan Java, JavaFX, dan MySQL.

## 🚀 Fitur Utama

- Login pengguna (dosen)
- Reservasi kelas berdasarkan tanggal dan waktu
- Melihat daftar reservasi aktif
- Terhubung ke database MySQL
- Antarmuka menggunakan JavaFX

## 🛠 Teknologi

- Java (JDK 11+)
- JavaFX
- MySQL
- Visual Studio Code / NetBeans
